#  🚀 İçerik

Üniversite bitirme projesi için Kütüphane Otomasyonu

Sisteme eklenilecek kitapların **ISBN** numaraları ile kayıt edildiği, kayıtlı kitapların üyelere istenilen tarih aralığında kiralanabileceği ve zamanında teslim etmeyen üyelere ceza uygulanacağı basit bir kütüphane otomasyonu

#  ✎ Güncelleme Notları

1.0.1 Güncellemesi
- Onay Kutuları Eklendi.
- TCNO,ISBN numaralarının sınırlandırılması
- Cinsiyet seçilmediğindeki bug kalktı

1.0.2 Güncellemesi
- Ceza sistemi eklendi(geçen her gün için 10TL ceza)

1.0.3 Güncellemesi
- Geçmiş Kiralıklar eklendi
- Ödeme Paneli eklendi
- AForge ve ZXing Eklentileri sayesinde kamera ile ISBN Kodu okuma
  (Bilgisayar Kamerası yerine DroidCam yazılımı ile Telefon ekranı tercih edilir)
  
1.0.4 Güncellemesi
- Google Books API ile ISBN Numarası girilip bilgileri oto yükleme eklendi
  (ApiKey süresizdir ve kullanılabilir)
- ISBNDB API ile ISBN Numarası girilip bilgileri oto yükleme eklendi
  (ApiKey 25.05.2024'de sona erer ve api kullanılamaz.)
  
1.0.5 Güncellemesi
- Rapor Paneli eklendi
- Ödemeleri ve Kiralıkları excel dosyasına aktarma mümkün kılındı
- Performans iyileştirmesi için veritabanından geleecek veriler
kullanıcın "Hepsini Göster" CheckBox'ı ile aktive olabilir hale
getirildi. Arama Kutularına veri girildiğinde de veriler yüklenebilir
  
#  👉🏼 Ekstra Eklenebilecek Özellikler

- ✅ - Geçmiş Kiralananlar veritabanı oluşturulabilir
- ✅ - Aylık Rapor
- ✅ - Google Books API bağlantısı ile oto bilgileri yerleştirme
- Teslim Tarihinin kiralandıktan sonra güncellenebilmesi
